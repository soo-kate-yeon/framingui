# playground-web

The FramingUI web app and hosted catalog.

This package powers the public site, hosted MCP metadata endpoints, and internal app surfaces used by the MCP server.

## Local Development

```bash
pnpm install
pnpm --filter playground-web dev
```

## Scope

- public documentation pages
- hosted theme, component, template, and example catalog routes
- product marketing and account surfaces

## Note

This package is application infrastructure, not a public npm package.
