# 컴포넌트 API 참조

이 참조는 props, 변형 및 사용 패턴과 함께 모든 FramingUI 컴포넌트를 문서화합니다. 인간 개발자와 AI 코딩 도구 모두를 위해 설계되었습니다.

## Button 컴포넌트

사용자 작업을 위한 주요 인터랙티브 요소입니다.

### Import

```tsx
import { Button } from 'framingui';
```

### Props

| Prop | 타입 | 기본값 | 설명 |
|------|------|--------|------|
| `variant` | `'primary' \| 'secondary' \| 'ghost' \| 'danger'` | `'primary'` | 시각적 스타일 변형 |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | 버튼 크기 |
| `disabled` | `boolean` | `false` | 버튼 상호작용 비활성화 |
| `loading` | `boolean` | `false` | 로딩 스피너 표시 |
| `fullWidth` | `boolean` | `false` | 버튼을 전체 너비로 확장 |
| `onClick` | `() => void` | - | 클릭 핸들러 |
| `children` | `ReactNode` | - | 버튼 내용 |

### 변형

**Primary** (기본값):
```tsx
<Button variant="primary">제출</Button>
```

**Secondary**:
```tsx
<Button variant="secondary">취소</Button>
```

**Ghost** (최소 스타일링):
```tsx
<Button variant="ghost">더 알아보기</Button>
```

**Danger** (파괴적 작업):
```tsx
<Button variant="danger">삭제</Button>
```

### 크기

```tsx
<Button size="sm">작음</Button>
<Button size="md">중간</Button>
<Button size="lg">큼</Button>
```

### 상태

**로딩**:
```tsx
<Button loading>제출 중...</Button>
```

**비활성화**:
```tsx
<Button disabled>제출</Button>
```

**전체 너비**:
```tsx
<Button fullWidth>계속</Button>
```

### 사용된 디자인 토큰

- 배경: `--color-interactive-primary`
- 호버: `--color-interactive-hover`
- 패딩: `--spacing-component-sm`, `--spacing-component-md`
- 모서리 반경: `--border-radius-md`

## Card 컴포넌트

관련 콘텐츠를 그룹화하기 위한 컨테이너입니다.

### Import

```tsx
import { Card } from 'framingui';
```

### Props

| Prop | 타입 | 기본값 | 설명 |
|------|------|--------|------|
| `variant` | `'default' \| 'outlined' \| 'elevated'` | `'default'` | 카드 스타일 변형 |
| `padding` | `'sm' \| 'md' \| 'lg'` | `'md'` | 내부 패딩 |
| `hoverable` | `boolean` | `false` | 호버 효과 추가 |
| `children` | `ReactNode` | - | 카드 내용 |

### 변형

**Default** (채워진 배경):
```tsx
<Card>
  <h3>카드 제목</h3>
  <p>카드 내용이 여기에 표시됩니다</p>
</Card>
```

**Outlined** (테두리만):
```tsx
<Card variant="outlined">
  <h3>아웃라인 카드</h3>
</Card>
```

**Elevated** (그림자 포함):
```tsx
<Card variant="elevated">
  <h3>입체 카드</h3>
</Card>
```

### 인터랙티브 카드

```tsx
<Card hoverable onClick={() => navigate('/details')}>
  <h3>클릭 가능한 카드</h3>
  <p>클릭하여 세부정보 보기</p>
</Card>
```

### 사용된 디자인 토큰

- 배경: `--color-surface-elevated`
- 테두리: `--color-border-subtle`
- 그림자: `--shadow-card-default`, `--shadow-card-hover`
- 패딩: `--spacing-component-md`
- 모서리 반경: `--border-radius-md`

## Input 컴포넌트

폼 데이터 수집을 위한 텍스트 입력 필드입니다.

### Import

```tsx
import { Input } from 'framingui';
```

### Props

| Prop | 타입 | 기본값 | 설명 |
|------|------|--------|------|
| `type` | `'text' \| 'email' \| 'password' \| 'number'` | `'text'` | 입력 유형 |
| `label` | `string` | - | 입력 레이블 |
| `placeholder` | `string` | - | 플레이스홀더 텍스트 |
| `error` | `string` | - | 오류 메시지 |
| `disabled` | `boolean` | `false` | 입력 비활성화 |
| `required` | `boolean` | `false` | 필수로 표시 |
| `value` | `string` | - | 제어 값 |
| `onChange` | `(value: string) => void` | - | 변경 핸들러 |

### 기본 사용법

```tsx
<Input
  label="이메일"
  type="email"
  placeholder="you@example.com"
  required
/>
```

### 오류 상태

```tsx
<Input
  label="비밀번호"
  type="password"
  error="비밀번호는 최소 8자여야 합니다"
/>
```

### 제어 입력

```tsx
const [email, setEmail] = useState('');

<Input
  label="이메일"
  value={email}
  onChange={setEmail}
/>
```

### 사용된 디자인 토큰

- 테두리: `--color-border-default`
- 포커스 테두리: `--color-border-focus`
- 텍스트: `--color-text-primary`
- 플레이스홀더: `--color-text-tertiary`
- 오류: `--color-status-error`
- 패딩: `--spacing-component-sm`
- 모서리 반경: `--border-radius-md`

## Badge 컴포넌트

상태, 수량 또는 카테고리 표시를 위한 작은 레이블입니다.

### Import

```tsx
import { Badge } from 'framingui';
```

### Props

| Prop | 타입 | 기본값 | 설명 |
|------|------|--------|------|
| `variant` | `'default' \| 'success' \| 'warning' \| 'error' \| 'info'` | `'default'` | 상태 변형 |
| `size` | `'sm' \| 'md'` | `'md'` | 배지 크기 |
| `children` | `ReactNode` | - | 배지 내용 |

### 변형

```tsx
<Badge variant="default">초안</Badge>
<Badge variant="success">게시됨</Badge>
<Badge variant="warning">대기 중</Badge>
<Badge variant="error">실패</Badge>
<Badge variant="info">새로움</Badge>
```

### 수량 표시

```tsx
<Badge variant="info">새 메시지 3개</Badge>
```

### 사용된 디자인 토큰

- 배경: `--color-status-*` (변형에 따라 다름)
- 텍스트: `--color-text-inverse`
- 패딩: `--spacing-component-xs`
- 모서리 반경: `--border-radius-sm`

## Avatar 컴포넌트

사용자 프로필 이미지 또는 이니셜 표시입니다.

### Import

```tsx
import { Avatar } from 'framingui';
```

### Props

| Prop | 타입 | 기본값 | 설명 |
|------|------|--------|------|
| `src` | `string` | - | 이미지 URL |
| `alt` | `string` | - | 대체 텍스트 |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | 아바타 크기 |
| `initials` | `string` | - | 대체 이니셜 |

### 이미지 포함

```tsx
<Avatar src="/user.jpg" alt="홍길동" size="md" />
```

### 이니셜 포함

```tsx
<Avatar initials="홍길" size="md" />
```

### 크기

```tsx
<Avatar src="/user.jpg" size="sm" />
<Avatar src="/user.jpg" size="md" />
<Avatar src="/user.jpg" size="lg" />
```

### 사용된 디자인 토큰

- 크기: `--component-avatar-size-sm/md/lg`
- 모서리 반경: `--border-radius-full`
- 배경: `--color-interactive-primary`
- 텍스트: `--color-text-inverse`

## Modal 컴포넌트

집중된 상호작용을 위한 오버레이 대화 상자입니다.

### Import

```tsx
import { Modal } from 'framingui';
```

### Props

| Prop | 타입 | 기본값 | 설명 |
|------|------|--------|------|
| `open` | `boolean` | `false` | 가시성 제어 |
| `onClose` | `() => void` | - | 닫기 핸들러 |
| `title` | `string` | - | 모달 제목 |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | 모달 크기 |
| `children` | `ReactNode` | - | 모달 내용 |

### 기본 사용법

```tsx
const [open, setOpen] = useState(false);

<Modal
  open={open}
  onClose={() => setOpen(false)}
  title="작업 확인"
>
  <p>계속하시겠습니까?</p>
  <div className="flex gap-2 mt-4">
    <Button onClick={() => setOpen(false)} variant="secondary">
      취소
    </Button>
    <Button onClick={handleConfirm}>
      확인
    </Button>
  </div>
</Modal>
```

### 사용된 디자인 토큰

- 오버레이: `--color-surface-overlay`
- 배경: `--color-surface-elevated`
- 그림자: `--shadow-modal`
- 패딩: `--spacing-component-lg`
- 모서리 반경: `--border-radius-lg`

## Select 컴포넌트

옵션을 선택하기 위한 드롭다운 메뉴입니다.

### Import

```tsx
import { Select } from 'framingui';
```

### Props

| Prop | 타입 | 기본값 | 설명 |
|------|------|--------|------|
| `options` | `{ value: string; label: string }[]` | - | 사용 가능한 옵션 |
| `value` | `string` | - | 선택된 값 |
| `onChange` | `(value: string) => void` | - | 변경 핸들러 |
| `placeholder` | `string` | - | 플레이스홀더 텍스트 |
| `label` | `string` | - | 선택 레이블 |
| `error` | `string` | - | 오류 메시지 |

### 기본 사용법

```tsx
const [country, setCountry] = useState('');

<Select
  label="국가"
  placeholder="국가를 선택하세요"
  options={[
    { value: 'kr', label: '대한민국' },
    { value: 'us', label: '미국' },
    { value: 'jp', label: '일본' }
  ]}
  value={country}
  onChange={setCountry}
/>
```

### 사용된 디자인 토큰

- 테두리: `--color-border-default`
- 포커스 테두리: `--color-border-focus`
- 배경: `--color-surface-elevated`
- 텍스트: `--color-text-primary`
- 패딩: `--spacing-component-sm`
- 모서리 반경: `--border-radius-md`

## AI 코드 생성 패턴

### 폼 생성

**프롬프트:**
> "이메일과 비밀번호 입력, 그리고 제출 버튼이 있는 로그인 폼을 만들어줘"

**예상 출력:**
```tsx
export function LoginForm() {
  const [email, setEmail] = useState('');
  const [password, setPassword] = useState('');

  return (
    <form className="space-y-[var(--spacing-layout-sm)]">
      <Input
        type="email"
        label="이메일"
        placeholder="you@example.com"
        value={email}
        onChange={setEmail}
        required
      />
      <Input
        type="password"
        label="비밀번호"
        value={password}
        onChange={setPassword}
        required
      />
      <Button type="submit" fullWidth>
        로그인
      </Button>
    </form>
  );
}
```

### 대시보드 카드 생성

**프롬프트:**
> "아이콘과 큰 숫자를 포함한 사용자 수 통계 카드를 만들어줘"

**예상 출력:**
```tsx
export function StatsCard({ icon: Icon, label, value }: StatsCardProps) {
  return (
    <Card variant="elevated">
      <div className="flex items-center gap-[var(--spacing-component-sm)]">
        <Icon className="w-5 h-5 text-[var(--color-interactive-primary)]" />
        <p className="text-[var(--typography-caption-fontSize)]
                      text-[var(--color-text-secondary)]">
          {label}
        </p>
      </div>
      <p className="text-[var(--typography-heading-1-fontSize)]
                    font-[var(--typography-heading-1-fontWeight)]
                    text-[var(--color-text-primary)]
                    mt-[var(--spacing-component-sm)]">
        {value}
      </p>
    </Card>
  );
}
```

## AI 생성을 위한 모범 사례

1. **원시 HTML보다 항상 컴포넌트 사용** — `<button>` 대신 `<Button>` 선호
2. **디자인 토큰 참조** — 모든 스타일링에 `var(--token-name)` 사용
3. **접근성 패턴 따르기** — 레이블, ARIA 속성 포함
4. **의미론적 변형 사용** — 적절한 `variant` props 선택
