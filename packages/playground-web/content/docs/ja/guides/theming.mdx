# テーマ設定

FramingUIはtoken-firstデザインシステムです。組み込みテーマで開始し、必要なものをカスタマイズして変更を保存します。

## 組み込みテーマ

FramingUIは7つのプロダクション対応テーマを提供:

| テーマ | 最適 | 雰囲気 |
| --- | --- | --- |
| `classic-magazine` | 編集、ブログ、コンテンツサイト | 伝統的、読みやすい |
| `editorial-tech` | ドキュメント、ナレッジベース | クリーン、技術的 |
| `dark-boldness` | ダッシュボード、アナリティクス | ハイコントラスト、劇的 |
| `minimal-workspace` | 生産性ツール、SaaS | 集中、邪魔なし |
| `neutral-workspace` | エンタープライズ、ビジネスアプリ | プロフェッショナル、バランス |
| `pebble` | コンシューマーアプリ、ソーシャル | ソフト、フレンドリー |
| `square-minimalism` | 管理パネル、フィンテック | 幾何学的、コンパクト |

## 1. テーマ選択

アプリエントリーポイントでテーマCSSをimport:

```tsx
// app/layout.tsx(Next.js)
import '@framingui/tokens/themes/minimal-workspace.css';
import '@framingui/ui/styles';
```

## 2. Tokens変更

グローバルCSSファイルでトークンをオーバーライド:

```css
/* app/globals.css */

/* ブランドカラー */
:root {
  --bg-primary: oklch(0.56 0.18 257);            /* カスタム紫 */
  --bg-primary-foreground: oklch(0.98 0 0);      /* Primary上の白テキスト */
  --bg-accent: oklch(0.72 0.14 200);             /* アクセント青 */
}

/* タイポグラフィ */
:root {
  --font-sans: 'Inter', system-ui, sans-serif;
  --text-base: 15px;                              /* やや大きいbase */
  --leading-normal: 1.6;                          /* より多いline height */
}

/* Spacing & Radius */
:root {
  --radius-lg: 12px;                              /* より丸いコーナー */
  --spacing-4: 1.25rem;                           /* よりゆとりのあるスペーシング */
}
```

## 3. テーマ保存

### オプションA: Gitにコミット

テーマはCSS変数だけです。`globals.css`をリポジトリにコミット:

```bash
git add app/globals.css
git commit -m "feat: customize theme tokens"
```

## 4. Dark Mode設定

FramingUIテーマはクラスベース切り替えによるdark modeをサポート。

### 基本Dark Mode

`.dark`セレクタ内でトークンをオーバーライド:

```css
/* app/globals.css */

/* Light mode(デフォルト) */
:root {
  --bg-background: oklch(1 0 0);
  --bg-foreground: oklch(0.2 0 0);
  --bg-card: oklch(0.99 0 0);
  --border-default: oklch(0.89 0 0);
}

/* Dark mode */
.dark {
  --bg-background: oklch(0.16 0.01 260);
  --bg-foreground: oklch(0.96 0.01 260);
  --bg-card: oklch(0.2 0.01 260);
  --bg-card-foreground: oklch(0.96 0.01 260);
  --border-default: oklch(0.32 0.01 260);
  --bg-muted: oklch(0.24 0.01 260);
}
```

### Dark Modeトグル

```tsx
'use client';
import { useEffect, useState } from 'react';
import { Button } from '@framingui/ui';

export function ThemeModeToggle() {
  const [dark, setDark] = useState(false);

  useEffect(() => {
    const isDark = window.matchMedia('(prefers-color-scheme: dark)').matches;
    setDark(isDark);
  }, []);

  useEffect(() => {
    document.documentElement.classList.toggle('dark', dark);
    localStorage.setItem('theme-mode', dark ? 'dark' : 'light');
  }, [dark]);

  return (
    <Button variant="ghost" onClick={() => setDark(v => !v)}>
      {dark ? '🌙' : '☀️'}
    </Button>
  );
}
```
