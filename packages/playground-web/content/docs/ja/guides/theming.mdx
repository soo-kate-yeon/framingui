# Theming

FramingUI はトークン中心のデザインシステムです。まずは組み込みテーマを選び、必要なトークンだけ上書きする使い方がおすすめです。

## 組み込みテーマ（7種類）

| Theme ID | 用途 |
| --- | --- |
| `classic-magazine` | エディトリアルやコンテンツ中心のプロダクト |
| `editorial-tech` | ドキュメントやナレッジ系プロダクト |
| `dark-boldness` | コントラストが高いダーク系ダッシュボード |
| `minimal-workspace` | 集中しやすい生産性インターフェース |
| `neutral-workspace` | バランス重視の業務アプリ |
| `pebble` | やわらかく親しみやすいコンシューマー UI |
| `square-minimalism` | 幾何学ベースでコンパクトな業務 UI |

## テーマを適用する

アプリのルートでテーマ CSS を読み込みます。

```tsx
// app/layout.tsx (Next.js)
import '@framingui/tokens/themes/square-minimalism.css';
import '@framingui/ui/styles';
```

## トークンを上書きする

グローバル CSS に上書きを追加します。変数名は `--tekton-*` 名前空間を使ってください。

```css
/* app/globals.css */
:root {
  --tekton-bg-primary: oklch(0.56 0.18 257);
  --tekton-bg-primary-foreground: oklch(0.98 0 0);
  --tekton-radius-lg: 14px;
  --tekton-spacing-4: 1rem;
}
```

一部エリアだけ見た目を変える場合は、スコープ付き上書きが便利です。

```css
.pricing-surface {
  --tekton-bg-card: oklch(0.99 0.01 250);
  --tekton-border-default: oklch(0.88 0.02 250);
}
```

## ダークモード設定

クラスベースで切り替え、変える必要があるトークンだけ上書きします。

```css
:root {
  --tekton-bg-background: oklch(1 0 0);
  --tekton-bg-foreground: oklch(0.2 0 0);
}

.dark {
  --tekton-bg-background: oklch(0.16 0.01 260);
  --tekton-bg-foreground: oklch(0.96 0.01 260);
  --tekton-bg-card: oklch(0.2 0.01 260);
  --tekton-border-default: oklch(0.32 0.01 260);
}
```

```tsx
// app/providers.tsx
import { useEffect, useState } from 'react';

export function ThemeModeToggle() {
  const [dark, setDark] = useState(false);

  useEffect(() => {
    document.documentElement.classList.toggle('dark', dark);
  }, [dark]);

  return (
    <button onClick={() => setDark(v => !v)}>
      {dark ? 'Switch to Light' : 'Switch to Dark'}
    </button>
  );
}
```

## おすすめの進め方

1. まず近いベーステーマを選ぶ
2. ブランドカラーとタイポグラフィのトークンから上書きする
3. ライト/ダーク両方でコントラストを確認する
4. コンポーネントコードは変えず、見た目はトークンで制御する
