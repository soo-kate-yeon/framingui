# はじめに

5分でAIネイティブデザインシステムを起動して実行します。

## 1. FramingUIインストール

```bash
npm install @framingui/ui @framingui/tokens
```

または好みのパッケージマネージャーで:

```bash
pnpm add @framingui/ui @framingui/tokens
# または
yarn add @framingui/ui @framingui/tokens
```

## 2. テーマとスタイル追加

テーマを選択し、ベーススタイルと一緒にアプリルートでimportします。

```tsx
// app/layout.tsx(Next.js App Router)
import '@framingui/tokens/themes/minimal-workspace.css';
import '@framingui/ui/styles';

export default function RootLayout({ children }) {
  return (
    <html lang="ja">
      <body>{children}</body>
    </html>
  );
}
```

```tsx
// pages/_app.tsx(Next.js Pages Router)
import '@framingui/tokens/themes/minimal-workspace.css';
import '@framingui/ui/styles';

export default function App({ Component, pageProps }) {
  return <Component {...pageProps} />;
}
```

```tsx
// main.tsx(Vite/React)
import '@framingui/tokens/themes/minimal-workspace.css';
import '@framingui/ui/styles';
import './index.css';
```

## 3. 最初のコンポーネントを使用

```tsx
import { Button } from '@framingui/ui';

export default function HomePage() {
  return (
    <div className="flex flex-col items-center justify-center min-h-screen gap-4">
      <h1 className="text-4xl font-bold">FramingUIへようこそ</h1>
      <div className="flex gap-3">
        <Button>Primary Action</Button>
        <Button variant="outline">Secondary</Button>
      </div>
    </div>
  );
}
```

## 4. MCP接続(オプション)

Claude Code、Cursor、またはWindsurfがデザインシステムトークンとコンポーネントを理解できるようにします。

```bash
npx @framingui/mcp-server init
```

プロジェクトルートに`.mcp.json`が作成されます。AIツールを再起動して質問:

> "すべてのFramingUIテーマをリストアップして"

## トラブルシューティング

### スタイルが表示されない?

- テーマCSSが`@framingui/ui/styles`の**前に**importされているか確認
- Tailwind configがFramingUIパスを含むか確認(`init`で自動設定)
- 設定変更後にdev serverを再起動

### TypeScriptエラー?

```bash
npm install -D @types/react @types/react-dom
```

### コンポーネントが見つからない?

インストール確認:

```bash
npm list @framingui/ui
```

ない場合は再インストール:

```bash
npm install @framingui/ui @framingui/tokens
```
