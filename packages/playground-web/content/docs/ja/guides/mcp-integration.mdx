# MCP統合

AIツールにデザインシステムを理解させます。FramingUI MCPを使うと、Claude Code、Cursor、Windsurfが一貫性のあるトークンベースUIコードを生成します。

## MCPとは?

**Model Context Protocol(MCP)**はAIツールが外部コンテキストとツールにアクセスできるようにするオープン標準です。FramingUI MCPが提供:

- 🎨 Design token情報
- 🧩 コンポーネントAPIリファレンス
- 🎭 利用可能なテーマとトークン
- 🔍 トークン検索と推奨

## クイックセットアップ

プロジェクトルートで実行:

```bash
npx @framingui/mcp-server init
```

このコマンドは:
1. `.mcp.json`設定を作成
2. FramingUIパスを含むようにTailwind configを更新
3. `@framingui/ui/styles`がimport可能か確認
4. 必要に応じて依存関係をインストール

## AIツールに接続

3つのツールすべてがプロジェクトルートの`.mcp.json`を読み取ります:

```json
{
  "mcpServers": {
    "framingui": {
      "type": "stdio",
      "command": "npx",
      "args": ["-y", "@framingui/mcp-server"]
    }
  }
}
```

### Claude Code

1. プロジェクトルートで`npx @framingui/mcp-server init`実行
2. Claude Codeでプロジェクトを開く
3. Claude Code再起動(Cmd/Ctrl + Shift + P → "Reload Window")
4. 確認: "利用可能なFramingUIテーマをリストアップして"と質問

### Cursor

1. ワークスペースルートに`.mcp.json`があるか確認
2. Cursorでプロジェクトを開く
3. Cursor再起動またはウィンドウリロード(Cmd/Ctrl + Shift + P → "Reload Window")
4. 確認: "FramingUIが提供するdesign tokensは何?"と質問

### Windsurf

1. プロジェクトルートに`.mcp.json`があるか確認
2. Windsurfでプロジェクトを開く
3. Windsurf再起動
4. 確認: "FramingUIコンポーネントリストを見せて"と質問

## 実際の使用例

### 例1: 一貫したUI生成

**プロンプト:**
> "タイトル、価格、機能リスト、CTAボタン付きのpricing cardコンポーネントをFramingUIで作って"

**MCPなし(❌ 一貫性なし):**
```tsx
export function PricingCard() {
  return (
    <div className="bg-white border border-gray-200 rounded-lg p-6">
      <h3 className="text-2xl font-bold text-gray-900">Pro Plan</h3>
      <p className="text-4xl font-bold text-blue-600 mt-2">$29<span className="text-base font-normal text-gray-500">/mo</span></p>
      <button className="mt-6 w-full bg-blue-600 text-white py-2 px-4 rounded-md hover:bg-blue-700">
        Get Started
      </button>
    </div>
  );
}
```

**MCPあり(✅ トークン使用):**
```tsx
import { Button, Card } from '@framingui/ui';

export function PricingCard() {
  return (
    <Card className="p-6">
      <h3 className="text-2xl font-bold text-foreground">Pro Plan</h3>
      <p className="text-4xl font-bold text-primary mt-2">
        $29<span className="text-base font-normal text-muted-foreground">/mo</span>
      </p>
      <Button className="mt-6 w-full">Get Started</Button>
    </Card>
  );
}
```

## トラブルシューティング

### MCPが検出されない

**`.mcp.json`の場所確認:**
```bash
ls -la .mcp.json
```

プロジェクトルート(`package.json`と同じレベル)にあるべきです。

**AIツールを再起動** `.mcp.json`作成/更新後。
