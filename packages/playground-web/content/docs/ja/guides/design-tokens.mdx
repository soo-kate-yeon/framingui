# Design Tokens

Design tokensはデザインシステムの唯一のsource of truthです。すべての場所に`#3B82F6`をハードコードする代わりに`--color-primary`を使用し、AIがこれを理解できます。

## Design Tokensとは?

Design tokensは視覚的デザイン決定を表す名前付き変数です:

```css
/* これの代わり ❌ */
.button {
  background: #3B82F6;
  padding: 12px 24px;
  border-radius: 8px;
  font-size: 14px;
}

/* Tokensを使用 ✅ */
.button {
  background: var(--bg-primary);
  padding: var(--spacing-3) var(--spacing-6);
  border-radius: var(--radius-md);
  font-size: var(--text-sm);
}
```

`--bg-primary`を変更すると、すべてのコンポーネントが自動更新されます。

## AIがTokensを必要とする理由

Claude Codeのような AIツールはUIコードを生成します。トークンなしではカラーとスペーシングをハードコードし、一貫性のない保守不可能なコードを作成します。

トークンがあれば、AIは:
- ✅ ブランドと即座に一致するUIを生成
- ✅ 生成されたすべてのコードで一貫性を維持
- ✅ コンポーネント書き直しなしでデザイン変更に対応
- ✅ セマンティック意図を理解("primary action" vs "#3B82F6")

## Tokenカテゴリ

### 1. Colors

Light/dark modeに適応するセマンティックカラートークン:

```css
--bg-background       /* ページ背景 */
--bg-foreground       /* テキストカラー */
--bg-primary          /* ブランドprimary */
--bg-primary-foreground /* Primary上のテキスト */
--bg-destructive      /* 危険アクション */
--bg-muted            /* 控えめな背景 */
--border-default      /* デフォルトボーダー */
--border-input        /* フォームボーダー */
```

### 2. Typography

```css
--font-sans           /* 本文フォントファミリー */
--font-mono           /* コードフォントファミリー */

--text-xs             /* 12px */
--text-sm             /* 14px */
--text-base           /* 16px */
--text-lg             /* 18px */
```

### 3. Spacing

```css
--spacing-1           /* 0.25rem (4px) */
--spacing-2           /* 0.5rem (8px) */
--spacing-3           /* 0.75rem (12px) */
--spacing-4           /* 1rem (16px) */
--spacing-6           /* 1.5rem (24px) */
```

## コードでTokensを使用

### CSSで

```css
.card {
  background: var(--bg-card);
  border: 1px solid var(--border-default);
  border-radius: var(--radius-lg);
  padding: var(--spacing-6);
  box-shadow: var(--shadow-sm);
}
```

### Tailwindで(FramingUI経由)

FramingUIはトークンをTailwindユーティリティにマッピング:

```tsx
<div className="bg-card border border-default rounded-lg p-6 shadow-sm">
  <h2 className="text-xl font-semibold text-foreground">Card Title</h2>
  <p className="text-sm text-muted-foreground">Description</p>
</div>
```

## Tokensの上書き

グローバルCSSですべてのトークンを上書きできます:

```css
/* app/globals.css */
:root {
  --bg-primary: oklch(0.56 0.18 257);       /* カスタムブランドカラー */
  --font-sans: 'Inter', system-ui, sans-serif; /* カスタムフォント */
  --radius-lg: 12px;                         /* より丸いコーナー */
  --spacing-4: 1.25rem;                      /* よりゆとりのあるスペーシング */
}
```

高度なカスタマイズは[テーマ設定](/docs/ja/guides/theming)を参照してください。
