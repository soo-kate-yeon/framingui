# クイックスタート

長い説明は省いて、すぐ使える手順だけまとめます。

## 1) MCP サーバーを初期化

最速で始めるなら次の 1 行です。

```bash
npx @framingui/mcp-server init
```

このコマンドで次を自動設定します。

- プロジェクト種別の検出（Next.js / Vite）
- 必要な依存関係のインストール
- Tailwind 設定
- MCP 連携設定

## 2) AI ツールを再起動

`init` 完了後、Claude Code / Cursor / Windsurf など MCP 対応ツールを再起動します。

## 3) すぐに作ってみる

自然言語でそのまま指示できます。

- 「square-minimalism テーマでログインページを作って」
- 「editorial-tech スタイルでサイドバー付きダッシュボードを作って」
- 「pebble トークンで 3 列の料金表を追加して」
