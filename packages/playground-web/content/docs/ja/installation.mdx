# インストール

使い慣れたパッケージマネージャーで導入できます。まず動かしてから調整する流れがおすすめです。

## パッケージ構成

- `@framingui/core`: トークン生成、テーマ解析、OKLCH ユーティリティ
- `@framingui/ui`: テーマ対応の React コンポーネント 30+
- `@framingui/tokens`: 元デザイントークン（カラー、タイポ、スペーシング）
- `@framingui/mcp-server`: AI ツール連携用 MCP サーバー

## 推奨インストール（最速）

```bash
npx @framingui/mcp-server init
```

プロジェクト検出から設定まで一度で完了します。

## 手動インストール

### npm

```bash
npm install @framingui/ui @framingui/core
```

### pnpm

```bash
pnpm add @framingui/ui @framingui/core
```

### yarn

```bash
yarn add @framingui/ui @framingui/core
```

## Tailwind 設定

`tailwind.config.ts` に FramingUI のパスを追加します。

```ts
import type { Config } from 'tailwindcss';

const config: Config = {
  content: [
    './app/**/*.{js,ts,jsx,tsx}',
    './node_modules/@framingui/ui/**/*.{js,ts,jsx,tsx}',
  ],
  plugins: [require('tailwindcss-animate')],
};

export default config;
```

## スタイルの import

グローバル CSS に次を追加します。

```css
@import '@framingui/ui/styles';
```

## Peer Dependencies

`@framingui/ui` には次の依存関係が必要です。

```text
react >= 18.0.0
react-dom >= 18.0.0
tailwind-merge >= 2.0.0
class-variance-authority >= 0.7.0
clsx >= 2.0.0
lucide-react >= 0.400.0
```
