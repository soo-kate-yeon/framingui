# Input

`Input` は FramingUI デザインシステムの基本テキストフィールドです。標準の HTML input 属性をすべて利用できます。

## Import

```tsx
import { Input } from '@framingui/ui';
```

## Props

| Prop | Type | Default | 説明 |
| --- | --- | --- | --- |
| `type` | `string` | ブラウザ既定 (`text`) | 入力タイプ（`email`、`password`、`number` など） |
| `className` | `string` | `undefined` | 追加ユーティリティクラス |
| `disabled` | `boolean` | `false` | 無効状態 |
| `required` | `boolean` | `false` | 必須入力 |
| `aria-invalid` | `boolean \| 'true' \| 'false'` | `false` | 支援技術向けのバリデーション状態 |
| `aria-describedby` | `string` | `undefined` | 補足文やエラー文との関連付け |
| `...props` | `React.InputHTMLAttributes<HTMLInputElement>` | - | ネイティブ input 属性 |

## Usage Examples

```tsx
import { Input, Label } from '@framingui/ui';

export function InputExamples() {
  return (
    <div className="grid gap-4 max-w-sm">
      <div className="grid gap-2">
        <Label htmlFor="email">Email</Label>
        <Input id="email" type="email" placeholder="you@company.com" />
      </div>

      <div className="grid gap-2">
        <Label htmlFor="password">Password</Label>
        <Input id="password" type="password" placeholder="••••••••" />
      </div>

      <Input type="search" placeholder="Search" />
    </div>
  );
}
```

## Validation Example

```tsx
import { useState } from 'react';
import { Input, Label } from '@framingui/ui';

export function EmailField() {
  const [email, setEmail] = useState('');
  const isInvalid = email.length > 0 && !/^[^\s@]+@[^\s@]+\.[^\s@]+$/.test(email);

  return (
    <div className="grid gap-2 max-w-sm">
      <Label htmlFor="signup-email">Work email</Label>
      <Input
        id="signup-email"
        type="email"
        value={email}
        onChange={(e) => setEmail(e.target.value)}
        aria-invalid={isInvalid}
        aria-describedby={isInvalid ? 'signup-email-error' : undefined}
        className={isInvalid ? 'border-[var(--tekton-bg-destructive)]' : undefined}
      />
      {isInvalid && (
        <p id="signup-email-error" className="text-sm text-[var(--tekton-bg-destructive)]">
          Enter a valid email address.
        </p>
      )}
    </div>
  );
}
```

## Notes

- 入力欄には必ずラベルを付けてください。
- `email`、`tel`、`url` などの意味に合った型を使うと、ブラウザとモバイルでの入力体験が良くなります。
- アクセシブルな検証表示には `aria-invalid` と `aria-describedby` を使ってください。
