# Card

`Card` は、関連する情報と操作を 1 つにまとめるためのコンテナコンポーネントです。

## Import

```tsx
import {
  Card,
  CardHeader,
  CardTitle,
  CardDescription,
  CardContent,
  CardFooter,
} from '@framingui/ui';
```

## Props

### Card

| Prop | Type | Default | 説明 |
| --- | --- | --- | --- |
| `className` | `string` | `undefined` | 追加ユーティリティクラス |
| `children` | `React.ReactNode` | `undefined` | Card の本文 |
| `...props` | `React.HTMLAttributes<HTMLDivElement>` | - | ネイティブ div 属性 |

### CardHeader / CardContent / CardFooter

| Prop | Type | Default | 説明 |
| --- | --- | --- | --- |
| `className` | `string` | `undefined` | レイアウトと余白の調整 |
| `children` | `React.ReactNode` | `undefined` | セクション内容 |
| `...props` | `React.HTMLAttributes<HTMLDivElement>` | - | ネイティブ div 属性 |

### CardTitle / CardDescription

| Prop | Type | Default | 説明 |
| --- | --- | --- | --- |
| `className` | `string` | `undefined` | タイポグラフィ調整 |
| `children` | `React.ReactNode` | `undefined` | テキスト内容 |
| `...props` | `React.HTMLAttributes<...>` | - | ネイティブ見出し/段落属性 |

## Usage Example

```tsx
import {
  Card,
  CardHeader,
  CardTitle,
  CardDescription,
  CardContent,
  CardFooter,
  Button,
  Input,
  Label,
} from '@framingui/ui';

export function ProfileCard() {
  return (
    <Card className="w-full max-w-md">
      <CardHeader>
        <CardTitle>Edit Profile</CardTitle>
        <CardDescription>Update your account details.</CardDescription>
      </CardHeader>

      <CardContent className="space-y-4">
        <div className="space-y-2">
          <Label htmlFor="name">Name</Label>
          <Input id="name" placeholder="Jane Doe" />
        </div>
      </CardContent>

      <CardFooter className="justify-end gap-2">
        <Button variant="outline">Cancel</Button>
        <Button>Save</Button>
      </CardFooter>
    </Card>
  );
}
```

## Sub-component Roles

- `CardHeader`: タイトルと説明エリア
- `CardContent`: メインコンテンツ
- `CardFooter`: アクションとステータス行

この構造をそろえると、画面全体の見通しがよくなります。
