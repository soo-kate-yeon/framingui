# Button

`Button` は FramingUI デザインシステムの基本アクション用コンポーネントです。

## Import

```tsx
import { Button } from '@framingui/ui';
```

## Props

| Prop | Type | Default | 説明 |
| --- | --- | --- | --- |
| `variant` | `'default' \| 'destructive' \| 'outline' \| 'secondary' \| 'ghost' \| 'link'` | `'default'` | 見た目のスタイル |
| `size` | `'default' \| 'sm' \| 'lg' \| 'icon'` | `'default'` | ボタンサイズ |
| `asChild` | `boolean` | `false` | Radix Slot を使って子要素として描画 |
| `className` | `string` | `undefined` | 追加ユーティリティクラス |
| `disabled` | `boolean` | `false` | 無効状態 |
| `...props` | `React.ButtonHTMLAttributes<HTMLButtonElement>` | - | ネイティブ button 属性 |

## Variants

- `default`: メインアクション
- `destructive`: 削除など破壊的操作
- `outline`: 目立たせすぎない補助アクション
- `secondary`: 補助の塗りボタン
- `ghost`: 情報量の多い UI 向けの最小アクション
- `link`: 段落やカード内のテキスト CTA

## Usage Examples

```tsx
import { Button } from '@framingui/ui';

export function ButtonExamples() {
  return (
    <div className="flex flex-wrap gap-3">
      <Button>Save</Button>
      <Button variant="destructive">Delete</Button>
      <Button variant="outline">Cancel</Button>
      <Button variant="secondary">Back</Button>
      <Button variant="ghost">More</Button>
      <Button variant="link">Learn more</Button>
      <Button size="sm">Small</Button>
      <Button size="lg">Large</Button>
      <Button size="icon" aria-label="Open menu">☰</Button>
    </div>
  );
}
```

### `asChild` Composition

```tsx
import Link from 'next/link';
import { Button } from '@framingui/ui';

export function LinkButton() {
  return (
    <Button asChild variant="outline">
      <Link href="/pricing">Go to pricing</Link>
    </Button>
  );
}
```

## Notes

- フォームでは `type="submit"` を明示してください。
- アイコンのみのボタンには `aria-label` を付けてください。
- セクションごとに主アクションは 1 つに絞ると分かりやすくなります。
