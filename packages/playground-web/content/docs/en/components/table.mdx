# Table

`Table` is a composable component for displaying structured data in rows and columns in the FramingUI design system.

## Import

```tsx
import {
  Table,
  TableHeader,
  TableBody,
  TableFooter,
  TableHead,
  TableRow,
  TableCell,
  TableCaption,
} from '@framingui/ui';
```

## Props

### Table

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `className` | `string` | `undefined` | Extra utility classes |
| `...props` | `React.HTMLAttributes<HTMLTableElement>` | - | Native table attributes |

| Prop | Type | Default | Description |

### 4. Data Grid with Sorting

    <Table>
      <TableHeader>
        <TableRow>
          <TableHead>
            <button onClick={() => setSortConfig({ key: 'name', direction: sortConfig.direction === 'asc' ? 'desc' : 'asc' })}>
              Name {sortConfig.key === 'name' && (sortConfig.direction === 'asc' ? '↑' : '↓')}

## Design Tokens

The Table component uses the following Tailwind utility classes and CSS custom properties:

- **Layout**: `w-full`, `overflow-auto`, `relative`
- **Typography**: `text-sm`, `text-left`, `font-medium`, `caption-bottom`
- **Spacing**: `p-4`, `px-4`, `h-12`, `mt-4`
- **Borders**: `border-b`, `border-t`
- **Colors**: Uses CSS variables for theming
  - `--tekton-bg-muted`: Muted background color
  - `--tekton-bg-muted-foreground`: Muted text color
  - `--tekton-border-default`: Default border color
- **Interactions**: `hover:bg-[var(--tekton-bg-muted)]/50`, `transition-colors`
- **States**: `data-[state=selected]:bg-[var(--tekton-bg-muted)]`

## Notes

- The table is automatically wrapped in a scrollable container for responsive behavior.
- Use `TableCaption` to provide context for screen readers and SEO.
- Add `className="text-right"` to `TableHead` and `TableCell` for right-aligned numeric columns.
- Use `data-state="selected"` on `TableRow` to highlight selected rows.
- Keep table headers concise and descriptive.
- For large datasets, consider implementing pagination or virtualization.
- Use semantic HTML attributes like `colSpan` and `rowSpan` when needed.
