# NavigationMenu

`NavigationMenu` is a horizontal navigation component with accessible dropdown menus, built on Radix UI primitives.

## Import

```tsx
import {
  NavigationMenu,
  NavigationMenuList,
  NavigationMenuItem,
  NavigationMenuTrigger,
  NavigationMenuContent,
  NavigationMenuLink,
  navigationMenuTriggerStyle,
} from '@framingui/ui';
```

## Basic Usage

```tsx
import {
  NavigationMenu,
  NavigationMenuList,
  NavigationMenuItem,
  NavigationMenuLink,
  navigationMenuTriggerStyle,
} from '@framingui/ui';

export function SimpleNav() {
  return (
    <NavigationMenu>
      <NavigationMenuList>
        <NavigationMenuItem>
          <NavigationMenuLink href="/" className={navigationMenuTriggerStyle()}>
            Home
          </NavigationMenuLink>
        </NavigationMenuItem>
        <NavigationMenuItem>
          <NavigationMenuLink href="/about" className={navigationMenuTriggerStyle()}>
            About
          </NavigationMenuLink>
        </NavigationMenuItem>
        <NavigationMenuItem>
          <NavigationMenuLink href="/contact" className={navigationMenuTriggerStyle()}>
            Contact
          </NavigationMenuLink>
        </NavigationMenuItem>
      </NavigationMenuList>
    </NavigationMenu>
  );
}
```

## Props API

### NavigationMenu

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `orientation` | `'horizontal' \| 'vertical'` | `'horizontal'` | Layout direction |
| `dir` | `'ltr' \| 'rtl'` | `'ltr'` | Text direction |
| `delayDuration` | `number` | `200` | Delay before showing dropdown (ms) |
| `skipDelayDuration` | `number` | `300` | Delay to skip when moving between items |
| `className` | `string` | `undefined` | Extra utility classes |

### NavigationMenuTrigger

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `className` | `string` | `undefined` | Extra utility classes |
| `disabled` | `boolean` | `false` | Disabled state |
| `children` | `ReactNode` | - | Trigger content |

### NavigationMenuContent

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `className` | `string` | `undefined` | Extra utility classes |
| `children` | `ReactNode` | - | Dropdown content |
| `forceMount` | `boolean` | `false` | Force content to mount |

### NavigationMenuLink

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `href` | `string` | - | Link destination |
| `active` | `boolean` | `false` | Active state |
| `asChild` | `boolean` | `false` | Render as child element |
| `className` | `string` | `undefined` | Extra utility classes |

```tsx
--tekton-text-popover-foreground // Dropdown text
--tekton-text-muted-foreground  // Secondary text
--tekton-border-default         // Border and indicator
```

## Best Practices

- Use `NavigationMenuTrigger` for dropdowns, `NavigationMenuLink` for direct links
- Add `aria-label` to icon-only navigation items for accessibility
- Keep dropdown content concise‚Äîmax 2-3 columns for mega menus
- Use `orientation="vertical"` for sidebar navigation
- Consider mobile responsiveness‚Äîuse Sheet for hamburger menus
- Mark current page with `data-active` attribute for visual feedback
- Keyboard navigation is built-in‚ÄîTab, Arrow keys, Enter, Escape all work
- Avoid nesting dropdowns more than 2 levels deep
