# Select

`Select` is an accessible dropdown selection component built on Radix UI with keyboard navigation and design token integration.

## Import

```tsx
import {
  Select,
  SelectContent,
  SelectItem,
  SelectTrigger,
  SelectValue,
  SelectGroup,
  SelectLabel,
  SelectSeparator,
} from '@framingui/ui';
```

## Props

### Select (Root)

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `value` | `string` | `undefined` | Controlled selected value |
| `defaultValue` | `string` | `undefined` | Uncontrolled default value |
| `onValueChange` | `(value: string) => void` | `undefined` | Callback when value changes |
| `disabled` | `boolean` | `false` | Disable entire select |
| `name` | `string` | `undefined` | Form field name |
| `required` | `boolean` | `false` | Required field validation |

### SelectTrigger

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `className` | `string` | `undefined` | Additional CSS classes |
| `id` | `string` | `undefined` | HTML id for label association |

### SelectValue

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `placeholder` | `string` | `undefined` | Text shown when no value selected |

### SelectItem

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `value` | `string` | **required** | Item value (must be unique) |
| `disabled` | `boolean` | `false` | Disable this item |
| `className` | `string` | `undefined` | Additional CSS classes |

```tsx

```

## Best Practices

- Use descriptive placeholders when no default value is set.
- Associate selects with labels using `htmlFor` and `id` for accessibility.
- Use `SelectGroup` and `SelectLabel` to organize many options into categories.
- Provide `disabled` state when options are temporarily unavailable.
- Keep option text concise (under 40 characters when possible).
- Use controlled state with `value` and `onValueChange` when the selection affects other UI.
- Add icons or color indicators for visual clarity (priority, status).
- For 2-3 options, consider using radio buttons or toggle groups instead.
