# Select

`Select`는 Radix UI 기반의 접근성 높은 드롭다운 선택 컴포넌트로, 키보드 네비게이션과 디자인 토큰이 통합되어 있다.

## Import

```tsx
import {
  Select,
  SelectContent,
  SelectItem,
  SelectTrigger,
  SelectValue,
  SelectGroup,
  SelectLabel,
  SelectSeparator,
} from '@framingui/ui';
```

## Props

### Select (Root)

| Prop | 타입 | 기본값 | 설명 |
| --- | --- | --- | --- |
| `value` | `string` | `undefined` | 제어 모드 선택 값 |
| `defaultValue` | `string` | `undefined` | 비제어 모드 기본 값 |
| `onValueChange` | `(value: string) => void` | `undefined` | 값 변경 시 콜백 |
| `disabled` | `boolean` | `false` | 전체 Select 비활성화 |
| `name` | `string` | `undefined` | 폼 필드 name 속성 |
| `required` | `boolean` | `false` | 필수 입력 검증 |

### SelectTrigger

| Prop | 타입 | 기본값 | 설명 |
| --- | --- | --- | --- |
| `className` | `string` | `undefined` | 추가 CSS 클래스 |
| `id` | `string` | `undefined` | Label 연결용 HTML id |

### SelectValue

| Prop | 타입 | 기본값 | 설명 |
| --- | --- | --- | --- |
| `placeholder` | `string` | `undefined` | 값이 없을 때 표시 텍스트 |

### SelectItem

| Prop | 타입 | 기본값 | 설명 |
| --- | --- | --- | --- |
| `value` | `string` | **필수** | 아이템 값 (고유해야 함) |
| `disabled` | `boolean` | `false` | 해당 항목 비활성화 |
| `className` | `string` | `undefined` | 추가 CSS 클래스 |

## 실전 시나리오

### 1. 국가 선택기

```tsx
import { Select, SelectContent, SelectItem, SelectTrigger, SelectValue } from '@framingui/ui';

export function CountrySelector() {
  return (
    <Select>
      <SelectTrigger className="w-[200px]">
        <SelectValue placeholder="국가 선택" />
      </SelectTrigger>
      <SelectContent>
        <SelectItem value="us">미국</SelectItem>
        <SelectItem value="uk">영국</SelectItem>
        <SelectItem value="kr">대한민국</SelectItem>
        <SelectItem value="jp">일본</SelectItem>
        <SelectItem value="ca">캐나다</SelectItem>
      </SelectContent>
    </Select>
  );
}
```

## 디자인 토큰

Select 컴포넌트는 일관된 테마를 위해 FramingUI 디자인 토큰을 사용한다:

```css
/* Trigger */
--tekton-bg-background
--tekton-border-input
--tekton-border-ring
--tekton-spacing-2, --tekton-spacing-3
--tekton-radius-md

/* Content/Dropdown */
--tekton-bg-popover
--tekton-bg-popover-foreground
--tekton-border-default
--tekton-radius-md

/* Items */
--tekton-bg-accent
--tekton-bg-accent-foreground
--tekton-radius-sm

/* Separators */
--tekton-bg-muted
```

## 사용 팁

- 기본값이 없을 때는 명확한 placeholder를 사용한다.
- 접근성을 위해 `htmlFor`와 `id`로 Label과 연결한다.
- 옵션이 많을 때는 `SelectGroup`과 `SelectLabel`로 카테고리를 나눈다.
- 일시적으로 사용 불가능한 옵션은 `disabled` 상태를 제공한다.
- 옵션 텍스트는 간결하게 유지한다 (가능하면 40자 이하).
- 선택값이 다른 UI에 영향을 줄 때는 `value`와 `onValueChange`로 제어 모드를 사용한다.
- 시각적 명확성을 위해 아이콘이나 색상 표시를 추가한다 (우선순위, 상태 등).
- 옵션이 2-3개라면 라디오 버튼이나 토글 그룹 사용을 고려한다.
