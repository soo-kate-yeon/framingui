# Table

`Table`은 FramingUI 디자인 시스템에서 구조화된 데이터를 행과 열로 표시하는 조합형 컴포넌트다.

## Import

```tsx
import {
  Table,
  TableHeader,
  TableBody,
  TableFooter,
  TableHead,
  TableRow,
  TableCell,
  TableCaption,
} from '@framingui/ui';
```

## Props

### Table

| Prop | 타입 | 기본값 | 설명 |
| --- | --- | --- | --- |
| `className` | `string` | `undefined` | 추가 유틸리티 클래스 |
| `...props` | `React.HTMLAttributes<HTMLTableElement>` | - | 기본 table 속성 |

| Prop | 타입 | 기본값 | 설명 |

## 디자인 토큰

Table 컴포넌트는 다음 Tailwind 유틸리티 클래스와 CSS 커스텀 속성을 사용한다:

- **레이아웃**: `w-full`, `overflow-auto`, `relative`
- **타이포그래피**: `text-sm`, `text-left`, `font-medium`, `caption-bottom`
- **여백**: `p-4`, `px-4`, `h-12`, `mt-4`
- **테두리**: `border-b`, `border-t`
- **색상**: 테마를 위한 CSS 변수 사용
  - `--tekton-bg-muted`: 약한 배경색
  - `--tekton-bg-muted-foreground`: 약한 텍스트 색상
  - `--tekton-border-default`: 기본 테두리 색상
- **인터랙션**: `hover:bg-[var(--tekton-bg-muted)]/50`, `transition-colors`
- **상태**: `data-[state=selected]:bg-[var(--tekton-bg-muted)]`

## 사용 팁

- 테이블은 반응형 동작을 위해 자동으로 스크롤 가능한 컨테이너로 감싸진다.
- 스크린 리더와 SEO를 위해 `TableCaption`으로 문맥을 제공하는 편이 좋다.
- 숫자 열을 오른쪽 정렬하려면 `TableHead`와 `TableCell`에 `className="text-right"`를 추가한다.
- 선택된 행을 강조하려면 `TableRow`에 `data-state="selected"`를 사용한다.
- 테이블 헤더는 간결하고 설명적으로 유지한다.
- 대용량 데이터셋에는 페이지네이션이나 가상화 구현을 고려한다.
- 필요할 때 `colSpan`이나 `rowSpan` 같은 시맨틱 HTML 속성을 사용한다.
