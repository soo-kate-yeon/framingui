# MCP 통합

AI 도구가 디자인 시스템을 이해하도록 합니다. FramingUI MCP를 사용하면 Claude Code, Cursor, Windsurf가 일관성 있고 토큰 기반 UI 코드를 생성합니다.

## MCP란?

**Model Context Protocol (MCP)**는 AI 도구가 외부 컨텍스트와 도구에 접근할 수 있도록 하는 오픈 표준입니다. FramingUI MCP 제공:

- 🎨 Design token 정보
- 🧩 컴포넌트 API 참조
- 🎭 사용 가능한 테마 및 토큰
- 🔍 토큰 검색 및 추천

## 빠른 설정

프로젝트 루트에서 실행:

```bash
npx @framingui/mcp-server init
```

이 명령은:
1. `.mcp.json` 설정 생성
2. FramingUI 경로를 포함하도록 Tailwind config 업데이트
3. `@framingui/ui/styles` import 가능 확인
4. 필요시 의존성 설치

## AI 도구에 연결

세 도구 모두 프로젝트 루트의 `.mcp.json`을 읽습니다:

```json
{
  "mcpServers": {
    "framingui": {
      "type": "stdio",
      "command": "npx",
      "args": ["-y", "@framingui/mcp-server"]
    }
  }
}
```

### Claude Code

1. 프로젝트 루트에서 `npx @framingui/mcp-server init` 실행
2. Claude Code에서 프로젝트 열기
3. Claude Code 재시작 (Cmd/Ctrl + Shift + P → "Reload Window")
4. 확인: "사용 가능한 FramingUI 테마 나열해줘" 질문

### Cursor

1. 워크스페이스 루트에 `.mcp.json` 있는지 확인
2. Cursor에서 프로젝트 열기
3. Cursor 재시작 또는 윈도우 리로드 (Cmd/Ctrl + Shift + P → "Reload Window")
4. 확인: "FramingUI가 제공하는 design tokens이 뭐야?" 질문

### Windsurf

1. 프로젝트 루트에 `.mcp.json` 있는지 확인
2. Windsurf에서 프로젝트 열기
3. Windsurf 재시작
4. 확인: "FramingUI 컴포넌트 목록 보여줘" 질문

## 실제 사용 예시

### 예시 1: 일관된 UI 생성

**프롬프트:**
> "제목, 가격, 기능 목록, CTA 버튼이 있는 pricing card 컴포넌트를 FramingUI로 만들어줘"

**MCP 없이 (❌ 일관성 없음):**
```tsx
export function PricingCard() {
  return (
    <div className="bg-white border border-gray-200 rounded-lg p-6">
      <h3 className="text-2xl font-bold text-gray-900">Pro Plan</h3>
      <p className="text-4xl font-bold text-blue-600 mt-2">$29<span className="text-base font-normal text-gray-500">/mo</span></p>
      <button className="mt-6 w-full bg-blue-600 text-white py-2 px-4 rounded-md hover:bg-blue-700">
        Get Started
      </button>
    </div>
  );
}
```

**MCP 사용 (✅ 토큰 사용):**
```tsx
import { Button, Card } from '@framingui/ui';

export function PricingCard() {
  return (
    <Card className="p-6">
      <h3 className="text-2xl font-bold text-foreground">Pro Plan</h3>
      <p className="text-4xl font-bold text-primary mt-2">
        $29<span className="text-base font-normal text-muted-foreground">/mo</span>
      </p>
      <Button className="mt-6 w-full">Get Started</Button>
    </Card>
  );
}
```

## 문제 해결

### MCP가 감지되지 않음

**`.mcp.json` 위치 확인:**
```bash
ls -la .mcp.json
```

프로젝트 루트(`package.json`과 같은 레벨)에 있어야 합니다.

**AI 도구 재시작** `.mcp.json` 생성/업데이트 후.
