## Cursor에 디자인 토큰이 필요한 이유

Cursor의 AI는 코드 생성에 강력하지만 디자인 토큰 없이는:

- 색상 값을 추측합니다 (`bg-blue-500`, `#3B82F6`, `rgb(59, 130, 246)`)
- 일관성 없는 간격을 사용합니다 (`p-4`, `px-5`, `py-3`)
- 임의의 테두리 반경 값을 만듭니다 (`rounded-md`, `rounded-lg`)

FramingUI는 모든 디자인 결정에 대한 **단일 진실 공급원**을 Cursor에 제공하여 이 문제를 해결합니다.

## 설치

### 1단계: FramingUI 설치

```bash
# pnpm 사용 (권장)
pnpm add @framingui/ui @framingui/core

# npm 사용
npm install @framingui/ui @framingui/core

# yarn 사용
yarn add @framingui/ui @framingui/core
```

### 2단계: 디자인 토큰 생성

```bash
# 대화형 모드
npx framingui generate

# 또는 브랜드 색상 직접 지정
npx framingui generate --color "#FF6B35"
```

디자인 토큰이 포함된 `.framingui/` 디렉토리가 생성됩니다:

```
.framingui/
├── tokens.css          # CSS 커스텀 속성
├── tokens.json         # 도구용 JSON 형식
└── tailwind.config.js  # Tailwind 통합
```

### 3단계: 앱에서 토큰 임포트

```tsx
// app/layout.tsx (Next.js App Router)
import '@framingui/ui/styles/tokens.css';
import './globals.css';

export default function RootLayout({ children }: { children: React.ReactNode }) {
  return (
    <html lang="ko">
      <body>{children}</body>
    </html>
  );
}
```

## Cursor 설정

### `.cursorrules` 파일 생성

프로젝트 루트에 `.cursorrules` 파일을 생성하세요:

```
# FramingUI 디자인 시스템 규칙

## 컴포넌트 라이브러리
항상 @framingui/ui에서 UI 컴포넌트 임포트:
- Button, Input, Card, Form, Dialog, Sheet, Tabs
- Label, Checkbox, RadioGroup, Select, Switch
- Table, Avatar, Badge, Tooltip, Popover

## 디자인 토큰
모든 스타일링에 CSS 커스텀 속성 사용:

### 색상
- Primary: var(--tekton-bg-primary)
- Secondary: var(--tekton-bg-secondary)
- Destructive: var(--tekton-bg-destructive)
- Muted: var(--tekton-bg-muted)
- Accent: var(--tekton-bg-accent)
- Card: var(--tekton-bg-card)
- Popover: var(--tekton-bg-popover)

### 간격
- var(--tekton-spacing-1)부터 var(--tekton-spacing-8)까지
- Tailwind의 p-1, p-2 등 대신 사용

### 테두리 반경
- var(--tekton-radius-sm)
- var(--tekton-radius-md)
- var(--tekton-radius-lg)
- var(--tekton-radius-xl)

### 테두리
- var(--tekton-border-default)
- var(--tekton-border-input)
- var(--tekton-border-ring)

## 코드 생성 규칙
1. 색상을 하드코딩하지 않고 항상 var(--tekton-*) 사용
2. 로컬 파일이 아닌 @framingui/ui에서 컴포넌트 임포트
3. 색상 이름(blue, red)이 아닌 시맨틱 토큰 이름(primary, secondary) 사용
4. 원시 HTML 요소보다 FramingUI 컴포넌트 선호
5. 토큰을 참조하는 Tailwind 클래스 사용: bg-[var(--tekton-bg-primary)]

## 예제 컴포넌트
```tsx
import { Button, Card, CardHeader, CardTitle, CardContent } from '@framingui/ui';

export function ExampleCard() {
  return (
    <Card>
      <CardHeader>
        <CardTitle>예제</CardTitle>
      </CardHeader>
      <CardContent className="space-y-[var(--tekton-spacing-4)]">
        <p className="text-[var(--tekton-bg-muted-foreground)]">
          카드 내용
        </p>
        <Button variant="default">클릭하세요</Button>
      </CardContent>
    </Card>
  );
}
```
```

### VSCode 설정 (Cursor용)

`.vscode/settings.json`에 추가:

```json
{
  "editor.quickSuggestions": {
    "strings": true
  },
  "css.customData": [".vscode/css-custom-data.json"]
}
```

토큰 자동완성을 위해 `.vscode/css-custom-data.json` 생성:

```json
{
  "version": 1.1,
  "properties": [
    {
      "name": "--tekton-bg-primary",
      "description": "Primary 배경 색상"
    },
    {
      "name": "--tekton-bg-secondary",
      "description": "Secondary 배경 색상"
    },
    {
      "name": "--tekton-spacing-4",
      "description": "1rem 간격 (16px)"
    },
    {
      "name": "--tekton-radius-md",
      "description": "중간 테두리 반경"
    }
  ]
}
```

## Cursor 통합 테스트

### 테스트 1: 로그인 폼 생성

Cursor에게 요청:

```
이메일과 비밀번호 필드가 있는 로그인 폼을 FramingUI 컴포넌트로 만들어줘
```

예상 출력:

```tsx
import { Button, Input, Label, Card, CardHeader, CardTitle, CardContent } from '@framingui/ui';

export function LoginForm() {
  return (
    <Card className="w-full max-w-md">
      <CardHeader>
        <CardTitle>로그인</CardTitle>
      </CardHeader>
      <CardContent className="space-y-[var(--tekton-spacing-4)]">
        <div className="space-y-[var(--tekton-spacing-2)]">
          <Label htmlFor="email">이메일</Label>
          <Input id="email" type="email" placeholder="you@example.com" />
        </div>
        <div className="space-y-[var(--tekton-spacing-2)]">
          <Label htmlFor="password">비밀번호</Label>
          <Input id="password" type="password" />
        </div>
        <Button className="w-full" variant="default">
          로그인
        </Button>
      </CardContent>
    </Card>
  );
}
```

✅ **확인**: 하드코딩된 색상 없음, `var(--tekton-*)` 토큰 사용

### 테스트 2: 대시보드 카드 생성

Cursor에게 요청:

```
아이콘과 값을 표시하는 총 사용자 수 통계 카드 만들어줘
```

예상 출력:

```tsx
import { Card, CardContent } from '@framingui/ui';
import { Users } from 'lucide-react';

export function StatsCard() {
  return (
    <Card>
      <CardContent className="p-[var(--tekton-spacing-6)]">
        <div className="flex items-center space-x-[var(--tekton-spacing-4)]">
          <div className="p-[var(--tekton-spacing-3)] bg-[var(--tekton-bg-primary)]/10 rounded-[var(--tekton-radius-lg)]">
            <Users className="w-6 h-6 text-[var(--tekton-bg-primary)]" />
          </div>
          <div>
            <p className="text-sm text-[var(--tekton-bg-muted-foreground)]">총 사용자</p>
            <p className="text-2xl font-bold">12,345</p>
          </div>
        </div>
      </CardContent>
    </Card>
  );
}
```

✅ **확인**: FramingUI Card 사용, 디자인 토큰 참조

## 문제 해결

### 문제: Cursor가 토큰 없이 Tailwind 클래스 생성

**해결책**: `.cursorrules` 업데이트 후 명시적으로 요청:

```
FramingUI와 var(--tekton-*) 토큰을 사용하여 버튼 생성
```

### 문제: 토큰 자동완성 작동 안 함

**해결책**: `.vscode/css-custom-data.json` 추가 후 Cursor 재시작

### 문제: 컴포넌트를 찾을 수 없음

**해결책**: 패키지 설치 확인:

```bash
pnpm list @framingui/ui
```

## 고급: 커스텀 토큰 확장

프로젝트별 필요에 따라 FramingUI 토큰을 확장할 수 있습니다:

```css
/* globals.css */
@import '@framingui/ui/styles/tokens.css';

:root {
  /* 커스텀 토큰으로 확장 */
  --my-app-header-height: 64px;
  --my-app-sidebar-width: 280px;

  /* FramingUI 토큰 오버라이드 (선택사항) */
  --tekton-radius-xl: 16px;
}
```
