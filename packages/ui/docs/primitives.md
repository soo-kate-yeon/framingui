# UI Primitives

Internal reference for FramingUI primitive components.

## Purpose

This file exists for contributors who need a compact map of the primitive layer.

## Examples of Primitives

- `Button`
- `Input`
- `Label`
- `Badge`
- `Avatar`
- `Separator`
- `Checkbox`
- `RadioGroup`
- `Switch`
- `Textarea`
- `Select`

## Source of Truth

- component implementation in `packages/ui/src`
- MCP component catalog and `preview-component`
- public package README and site docs for user-facing examples

Do not keep large code dumps here.
