# UI Templates API

Internal reference for FramingUI template concepts.

## Current Position

Templates are inspiration and structure hints.
They are not the final production contract.

## In MCP Workflows

Templates should be treated as:

- optional hints from `list-screen-templates`
- deeper references from `preview-screen-template`
- non-authoritative inputs compared with validated screen definitions

## Keep in Sync With

- `packages/mcp-server/docs`
- template catalog implementation
- generated context fields such as `templateHints`
