## 2. サービスの性質

Tekton は次のようなデジタルサービスおよびデジタル資産を提供します。

- 月間ツールユニットを含む定期プラン
- トップアップまたは従量型の利用 entitlement
- レガシーテーマ、テンプレート、関連ダウンロード資産

アクセス権は購入後すぐに有効化され、利用も直ちに開始できる場合があるため、返金可否は実質的な使用またはダウンロードがすでに生じたかどうかに左右されます。

移行期間中に表示される shadow billing は参考情報であり、それ自体は請求ではありません。実際に購入または請求された金額のみ返金審査の対象となります。

## 3. 返金対象となる場合

次の場合、返金対象となる可能性があります。

| 状況             | 返金可否     | 条件                                                                      |
| ---------------- | ------------ | ------------------------------------------------------------------------- |
| 重複購入         | 可           | 同一アカウントまたは同一注文への重複請求                                  |
| 請求エラー       | 可           | 当社または Paddle の誤りで誤請求が発生した場合                            |
| 技術的障害       | 可           | 購入した entitlement を実質的に利用できず、合理的期間内に修正できない場合 |
| 初回プラン購入   | 条件付きで可 | 購入後 14 日以内かつツールユニットの実質消費前                            |
| トップアップ購入 | 条件付きで可 | 購入後 14 日以内かつ購入ユニットの実質消費前                              |
| レガシー資産購入 | 条件付きで可 | 購入後 14 日以内かつダウンロードまたは有効化前                            |

---

## 4. 返金できない場合

次の場合、原則として返金できません。

| 状況                                           | 返金可否 |
| ---------------------------------------------- | -------- |
| ツールユニットがすでに実質消費された場合       | 不可     |
| レガシー資産をダウンロードまたは有効化した場合 | 不可     |
| 実質利用後の単なる気が変わったという理由       | 不可     |
| 文書化された対応範囲外の互換性問題             | 不可     |
| サービスや資産を使うための知識不足             | 不可     |
| すでに終了した過去請求期間の料金               | 不可     |

## 5. 解約ポリシー

### 5.1 定期プラン

- 定期プランはいつでも解約できます。
- 法令で別段の定めがない限り、解約は現在の請求期間の終了時に反映されます。
- 解約により将来の更新は停止しますが、現在の請求期間について自動的に返金されるものではありません。

### 5.2 トップアップ

トップアップは一回限りの購入です。実質的に消費された後は返金できません。

### 5.3 レガシー購入

レガシーテーマまたはテンプレートの購入に基づくアクセス権は購入当時の条件に従って維持され、返金審査は本ポリシーに従います。
