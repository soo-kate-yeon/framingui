## 2. デジタル商品の性質

Tekton で販売する商品はすべて**デジタル商品**（React UI テンプレート、デザインアセット、コードパッケージ）です。デジタル商品の性質上、以下が原則です。

> **商品をダウンロードした後は、原則として返金できません。**

デジタル商品は物理商品と異なり返品できず、ダウンロード時点で商品の利用権が即時に提供されるためです。

## 3. 返金対象

### 3.1 返金できるケース

| 状況                           | 返金可否 | 条件                                             |
| ------------------------------ | -------- | ------------------------------------------------ |
| **ダウンロード前のキャンセル** | ✅ 可    | 購入後14日以内、かつ未ダウンロード               |
| **技術的不具合**               | ✅ 可    | 商品に重大な不具合があり、当社で解消できない場合 |
| **重複購入**                   | ✅ 可    | 同一商品を誤って重複決済した場合                 |

### 3.2 返金できないケース

| 状況                                               | 返金可否 |
| -------------------------------------------------- | -------- |
| ダウンロード済み商品（利用有無を問わず）           | ❌ 不可  |
| ダウンロード後の気が変わったという理由             | ❌ 不可  |
| ご利用プロジェクトとの互換性不足（不具合ではない） | ❌ 不可  |
| 商品を使うための知識不足                           | ❌ 不可  |
| 類似商品を他サービスで購入済み                     | ❌ 不可  |

## 4. サブスクリプション返金（Creator Pass）

### 4.1 解約ポリシー

- Creator Pass はいつでも解約できます
- 解約は現在の請求期間の終了時に反映されます
- 期間終了までは全テンプレートにアクセスできます

### 4.2 サブスクリプション返金ポリシー

| 状況                                             | 返金                                       |
| ------------------------------------------------ | ------------------------------------------ |
| 初回購入から14日以内に解約（未ダウンロード）     | ✅ 全額返金                                |
| 14日経過後、またはテンプレートをダウンロード済み | ❌ 返金不可（期間終了まで利用可）          |
| 更新日より前に解約した更新課金                   | 更新は実行されません（前期間の返金は不可） |

## 5. 返金申請の方法

### Step 1: ご連絡

以下の情報を添えて **soo.kate.yeon@gmail.com** までメールをお送りください。

- 件名: `Refund Request - [注文ID]`
- 登録メールアドレス
- 注文IDまたは取引参照番号
- 返金申請の理由

### Step 2: 審査

申請は**5営業日以内**に確認し、次のいずれかを案内します。

- 返金承認と処理の案内（対象の場合）
- 却下理由の案内（対象外の場合）
- 追加情報の依頼（必要な場合）

### Step 3: 返金処理

承認された返金は決済代行会社 **Paddle** を通じて処理され、通常は次の期間で反映されます。

- クレジットカード／デビットカード: 5〜10営業日
- PayPal: 3〜5営業日

## 6. チャージバック

問題解決のための連絡なしに、銀行または決済事業者へチャージバック（支払異議申立て）を行った場合:

- 調査中はアカウントを停止することがあります
- 不当なチャージバック（例: ダウンロード済み商品の利用後申立て）と判断した場合、当社は適切な対応を行う権利を留保します

チャージバックの前に、まず当社へご連絡ください。

---

## 7. 例外対応

次のような特別な事情では、個別判断で返金に応じる場合があります。

- 商品説明と実際の商品に重大な相違がある場合
- 長時間の障害で商品アクセスが困難だった場合
- その他、当社が妥当と判断する事情
