## 2. 서비스의 성격

Tekton은 다음과 같은 디지털 서비스 및 디지털 자산을 판매합니다.

- 월간 포함 툴 유닛이 있는 정기 플랜
- 탑업 또는 종량형 사용 entitlement
- 레거시 테마, 템플릿, 관련 다운로드 자산

접근 권한은 결제 후 즉시 부여될 수 있고, 사용도 즉시 시작될 수 있으므로 환불 가능 여부는 실제로 의미 있는 사용이나 다운로드가 이미 발생했는지에 따라 달라집니다.

전환 기간에 표시되는 shadow billing 화면은 참고용일 뿐이며, 실제로 구매되거나 청구된 금액에 대해서만 환불 심사가 가능합니다.

## 3. 환불 가능 사유

다음 경우 환불 대상이 될 수 있습니다.

| 상황             | 환불 가능 여부 | 조건                                                                               |
| ---------------- | -------------- | ---------------------------------------------------------------------------------- |
| 중복 구매        | 가능           | 동일 계정 또는 동일 주문에 대한 중복 청구                                          |
| 결제 오류        | 가능           | 당사 또는 Paddle의 오류로 잘못된 금액이 청구된 경우                                |
| 기술적 장애      | 가능           | 구매한 entitlement를 실질적으로 사용할 수 없고 합리적인 기간 내 수정되지 않는 경우 |
| 최초 플랜 구매   | 가능할 수 있음 | 구매 후 14일 이내이면서 툴 유닛이 실질적으로 소진되기 전                           |
| 탑업 구매        | 가능할 수 있음 | 구매 후 14일 이내이면서 구매한 툴 유닛이 실질적으로 소진되기 전                    |
| 레거시 자산 구매 | 가능할 수 있음 | 구매 후 14일 이내이면서 다운로드 또는 활성화 이전                                  |

## 4. 환불이 어려운 경우

다음의 경우에는 일반적으로 환불되지 않습니다.

| 상황                                       | 환불 가능 여부 |
| ------------------------------------------ | -------------- |
| 툴 유닛이 이미 실질적으로 사용된 경우      | 불가           |
| 레거시 자산을 다운로드했거나 활성화한 경우 | 불가           |
| 의미 있는 사용 후 단순 변심                | 불가           |
| 문서화된 지원 범위를 벗어난 호환성 문제    | 불가           |
| 서비스나 자산을 활용할 전문성 부족         | 불가           |
| 이미 종료된 이전 결제 기간에 대한 청구     | 불가           |

## 5. 취소 정책

### 5.1 정기 플랜

- 정기 플랜은 언제든지 취소할 수 있습니다.
- 취소는 관련 법령이 달리 요구하지 않는 한 현재 결제 기간 종료 시 적용됩니다.
- 취소는 향후 갱신을 중단하지만, 현재 결제 기간에 대한 자동 환불을 의미하지는 않습니다.

### 5.2 탑업

탑업은 일회성 구매입니다. 실질적으로 소진된 후에는 환불되지 않습니다.

### 5.3 레거시 구매

레거시 테마 또는 템플릿 구매에 따른 접근 권한은 구매 당시의 조건에 따라 유지되며, 환불 심사는 본 정책을 따릅니다.
