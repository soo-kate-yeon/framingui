## 2. 디지털 상품의 특성 및 청약철회 제한

Tekton에서 판매되는 모든 제품은 **디지털 콘텐츠**(React UI 템플릿, 디자인 에셋, 코드 패키지)입니다.

### 2.1 청약철회 제한 사유

「전자상거래 등에서의 소비자보호에 관한 법률」 제17조 제2항에 따라, 다음의 경우 청약철회가 제한됩니다:

> **디지털 콘텐츠의 경우, 이용자가 다운로드를 시작하면 청약철회(환불)가 불가능합니다.**

이는 디지털 상품은 물리적 상품처럼 "반품"할 수 없으며, 다운로드 시 상품의 전체 내용이 복제되기 때문입니다.

### 2.2 사전 고지 및 동의

당사는 결제 단계에서 다음과 같이 청약철회 제한에 대해 **별도로 고지**하고 있습니다:

- 결제 페이지에 "다운로드 시작 시 청약철회권이 제한됩니다" 문구 표시
- 다운로드 버튼 클릭 전 청약철회 제한 동의 확인

> **중요:** 다운로드 버튼을 클릭함으로써 귀하는 청약철회권 행사가 제한됨에 동의하는 것으로 간주됩니다.

## 3. 환불 자격

### 3.1 환불 가능

| 상황                          | 환불 가능 여부 | 조건                                                   |
| ----------------------------- | -------------- | ------------------------------------------------------ |
| **다운로드 전 취소**          | ✅ 가능        | 구매 후 7일 이내 요청 AND 다운로드 전                  |
| **기술적 결함**               | ✅ 가능        | 상품에 근본적인 결함이 있고 당사가 수정할 수 없는 경우 |
| **중복 구매**                 | ✅ 가능        | 동일 상품에 대한 실수로 인한 중복 결제                 |
| **상품 설명과 현저한 불일치** | ✅ 가능        | 상품이 설명과 현저히 다른 경우                         |

### 3.2 환불 불가

| 상황                                               | 환불 가능 여부 |
| -------------------------------------------------- | -------------- |
| 다운로드한 상품 (사용 여부 무관)                   | ❌ 불가        |
| 다운로드 후 마음이 바뀐 경우                       | ❌ 불가        |
| 귀하의 프로젝트와 호환되지 않음 (결함이 아닌 경우) | ❌ 불가        |
| 상품 사용을 위한 전문 지식 부족                    | ❌ 불가        |
| 다른 곳에서 유사한 상품을 이미 구매함              | ❌ 불가        |

## 4. 구독 환불 (Creator Pass)

### 4.1 취소 정책

- Creator Pass 구독은 언제든지 취소할 수 있습니다
- 취소는 현재 결제 기간 종료 시 적용됩니다
- 구독이 만료될 때까지 모든 템플릿에 대한 접근 권한이 유지됩니다

### 4.2 구독 환불 정책

| 상황                                       | 환불                                             |
| ------------------------------------------ | ------------------------------------------------ |
| 최초 구매 후 7일 이내 취소 (다운로드 없음) | ✅ 전액 환불                                     |
| 7일 이후 취소 또는 템플릿 다운로드 후      | ❌ 환불 불가; 기간 종료까지 접근 유지            |
| 갱신 요금 (갱신일 이전 취소된 경우)        | 갱신이 발생하지 않음; 이전 기간에 대한 환불 없음 |

## 6. 지불 거절 (Chargeback)

당사에 먼저 연락하지 않고 은행이나 결제 제공업체를 통해 지불 거절(분쟁)을 제기하는 경우:

- 조사가 완료될 때까지 계정이 정지될 수 있습니다
- 지불 거절이 부당한 것으로 판명되는 경우(예: 상품이 다운로드되어 사용된 경우), 당사는 적절한 구제 조치를 취할 권리를 보유합니다

지불 거절을 제기하기 전에 먼저 당사에 연락하여 문제를 해결하시기 바랍니다.
