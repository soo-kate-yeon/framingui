## 4. 제품 및 서비스

### 4.1 제품 설명

Tekton은 다음을 제공합니다:

- 프리미엄 React UI 템플릿 및 컴포넌트 라이브러리
- AI 기반 디자인 시스템 엔진
- WCAG 준수 디자인 토큰

### 4.2 배송 방식

모든 제품은 전자 다운로드를 통해 배송되는 디지털 상품입니다. 물리적 제품은 배송되지 않습니다.

### 4.3 업데이트

구매한 템플릿에는 해당 플랜에 명시된 기간 동안 업데이트가 포함됩니다:

- **Single Template / Double Package**: 구매일로부터 1년간 업데이트
- **Creator Pass**: 활성 구독 기간 동안 업데이트

## 5. 가격 및 결제

### 5.1 가격 플랜

| 플랜            | 가격           | 포함 내용                        |
| --------------- | -------------- | -------------------------------- |
| Single Template | 템플릿별 상이  | 템플릿 1개 + 1년 업데이트        |
| Double Package  | $99 (고정)     | 템플릿 2개 + 1년 업데이트        |
| Creator Pass    | $149/년 (고정) | 모든 템플릿 + 구독 기간 업데이트 |

**Single Template 가격:** 개별 템플릿은 범위와 수준에 따라 가격이 다르게 책정됩니다. 현재 가격은 구매 시점에 각 상품 페이지에서 확인할 수 있습니다.

### 5.2 결제 처리

모든 결제는 **Paddle**(당사의 결제 대행사)을 통해 처리됩니다. 구매 시 Paddle의 이용약관에도 동의하는 것으로 간주됩니다.

### 5.3 구독 조건

- Creator Pass 구독은 취소하지 않는 한 매년 자동 갱신됩니다
- 계정 설정을 통해 언제든지 취소할 수 있습니다
- 취소는 현재 결제 기간 종료 시 적용됩니다

### 5.4 통화 및 세금

가격은 USD로 표시됩니다. 해당 세금(부가가치세 포함)은 귀하의 위치에 따라 Paddle에서 계산하고 징수합니다.

## 6. 라이선스 부여

### 6.1 허용되는 사용

유효한 구매 시 당사는 귀하에게 다음을 위한 **비독점적, 양도 불가능한 라이선스**를 부여합니다:

- 무제한 개인 프로젝트에서 템플릿 사용
- 클라이언트 프로젝트에서 템플릿 사용 (최종 결과물에만)
- 귀하의 용도에 맞게 템플릿 수정

### 6.2 제한 사항

다음은 허용되지 **않습니다**:

- 템플릿 재판매, 재배포 또는 재라이선스 (원본 또는 수정본)
- 라이선스를 구매하지 않은 개인과 템플릿 공유
- 경쟁 템플릿/UI 키트 제품 생성을 위한 템플릿 사용
- 저작권 표시 제거 또는 변경

### 6.3 라이선스 범위

- **Single Template / Double Package**: 구매한 템플릿에 대한 라이선스는 영구적이며, 업데이트는 1년으로 제한
- **Creator Pass**: 모든 템플릿에 대한 접근은 활성 구독 기간에만 유효; 다운로드한 템플릿은 영구적으로 사용 가능
